* gh-open

GitHub remote url open command.

** Usage

   #+BEGIN_SRC sh
     gh-open ~/path/to/repo
   #+END_SRC

** Requirements

   - git command (in PATH)

** Install

   For Go User:
   #+BEGIN_SRC sh
     go get -u github.com/typester/gh-open
   #+END_SRC

   For others:

   [[https://github.com/typester/gh-open/releases][Downloads release binary and put it in your PATH]]
