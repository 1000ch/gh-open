# gh-open ![GitHub Actions Status](https://github.com/1000ch/gh-open/actions/workflows/go.yml/badge.svg)

GitHub remote url open command.

## Usage

```sh
gh-open ~/path/to/repo
```

## Requirements

- git command (in PATH)

## Install

```sh
go get -u github.com/typester/gh-open
```
